## Fastify Typescript Boilerplate


### Install Dependencies
```
npm install
```

### Start For Develop 
server start on `http://localhost:3000`
```
npm run dev
```

### Test
```
npm run test
```

### Build
```
npm run build
```

### Docker Compose
```
docker-compose -f docker-compose.production.yaml build
docker-compose -f docker-compose.production.yaml up
```
